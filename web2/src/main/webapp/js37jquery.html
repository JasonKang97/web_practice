자바스크립트 라이브러리 중 jQuery는 자바스크립트를 더 쉽게 사용하기 위해 만들어진 라이브러리다.
HTML에서 text, 이미지 등이 섞여 있을 경우, JS의 경우 데이터를 모두 불러온 후 실행한다. 반면 jQuery의 경우 text만 먼저 가져온 후 실행한다.
HTML 속 클라이언트 자바스크립트 작업을 간소화해 생산성을 향상시켜준다.
jQuery UI 플러그인을 사용할 수 있다.

$(document).ready(function(){
	실행문;
})
